前提概念

要弄明白一个前提概念：

内存泄露实际上的本质是观察一个对象的引用的对象是否存活。那么这个对象必须是要有具体的生命周期的，在Android 的体现就是四大组件。

ActivityLifecycleCallbacks

这个是一个Android Api 14以上允许在Application监听每一个Activity生命周期的回调方法。

  • 在onActivityDestoryed中，LeakCanary库基于这个生命回调，自动对Activity被结束之后，进行了后续Activity以及其引用对象是否被回收的情况。

  • refWatcher.watch对Activity进行具体的监听。
