Accessibility: Button & Image Demo

1. Button

A native <button> element. Android accessibility reports: android.widget.Button

Role: button → ClassName: android.widget.Button

2. Clickable Image

An <img> with onclick. Android accessibility reports: android.widget.ImageButton

Role: img + clickable → ClassName: android.widget.ImageButton

3. Non-clickable Image

A plain <img> without any click handler. Android accessibility reports: android.widget.Image

Role: img (not clickable) → ClassName: android.widget.Image
